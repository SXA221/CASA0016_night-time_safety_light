# CASA0016_night-time_safety_light
Final crit Arduino Uno code for CASA0016 project

# Description
This light sensor project uses:
- Light Dependent Resistor (LDR) to detect light levels
- Ultrasonic sensor to measure distance
- NeoPixel ring for visual feedback
- LED indicator

# Hardware
- Arduino Uno
- LDR and 220 Ohm resistor
- HC-SR04 ultrasonic range finder sensor
- 12-pixel NeoPixel ring
- Yellow LED

# Behaviour
- LED turns on when light level exceeds threshold
- NeoPixel ring lights up when an object is within range
